## Lasting Systems

A modern one-page experience dedicated to building technology that endures: reliable systems, meaningful progress, and security as stewardship. The design pairs a serious tone with subtle intensity to reflect an analytical, introverted logician who architects platforms that provide for others.

### Stack

- Next.js App Router (TypeScript)
- Tailwind CSS with custom gradients and typography
- Deployed for Vercel-ready environments

### Development

```bash
npm install
npm run dev
```

The app is available at `http://localhost:3000`.

### Production

```bash
npm run build
npm run start
```

### Linting

```bash
npm run lint
```
